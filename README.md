**Decoding Tamil Nadu Assembly Elections 2026**

A storytelling-first Data Analytics project created as part of the Codebasics Resume Project Challenge.

This project analyzes Tamil Nadu’s 2026 Assembly Election using publicly available Election Commission of India (ECI) data to uncover constituency-level competition, regional seat distribution patterns, and changing electoral competitiveness across the state.

The project follows a completely non-partisan and data-driven approach focused on election storytelling through dashboards, SQL analysis, and stakeholder presentation design.

**Project Objective**

The objective of this project is to identify and present meaningful election stories using only publicly available election data.

The analysis focuses on:

Regional seat distribution shifts

Constituency-level seat flips

Margin of victory and competitiveness trends

Rather than political commentary, the project emphasizes:

data storytelling

analytical clarity

dashboard usability

reproducibility

**Key Insights**

**1. Regional Seat Distribution Story**

Although Tamil Nadu’s six regions contributed a similar number of seats in both elections, the distribution of those seats shifted significantly between 2021 and 2026.

Several regions that previously showed concentrated seat patterns became more competitive, resulting in a more balanced and fragmented electoral map across the state.

**2. Constituency Flip Story**

Nearly 70% of constituencies changed winning parties between 2021 and 2026.

The analysis revealed large-scale constituency-level churn and widespread redistribution of seats across the state rather than isolated regional swings.

**3. Margin of Victory Story**

Electoral contests became noticeably tighter in 2026.

Average victory margins declined, narrow-margin victories increased significantly, and large-margin wins became less common, indicating rising electoral competitiveness across constituencies.
