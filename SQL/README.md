# Decoding Tamil Nadu Assembly Elections 2026

## About the Project

This project is a storytelling-first Data Analytics case study created as part of the **Codebasics Resume Project Challenge**.

The analysis uses publicly available **Election Commission of India (ECI)** data to explore how Tamil Nadu’s 2026 Assembly Elections evolved across regions, constituencies, and electoral competitiveness metrics.

The project focuses on uncovering:

* Regional seat distribution patterns
* Constituency-level seat shifts
* Margin of victory trends
* Electoral competitiveness across the state

The analysis remains completely **non-partisan**, data-driven, and focused only on observable election outcomes.
